# COMM061 NLP Coursework

Group project for Sentiment and Sarcasm classification
across English varieties using the BESSTIE dataset.

## Structure
notebooks/ - experiment notebooks  
  1 Load dataset
  2 Dataset analysis
  3 Baseline models
  4 Transformer models
  5 Cross variety experiments
  6 Evaluation
src/ - reusable python functions  
results/ - experiment outputs  
figures/ - plots used in report  
report/ - report drafts
